Introduction

JavaScript is a cross-platform, object-oriented scripting language. It is a small and lightweight language. Inside a host environment (for example, a web browser), JavaScript can be connected to the objects of the environment to provide programmatic control over them.

JavaScript contains a standard library of objects, such as Array, Date, and Math, and a core set of language elements such as operators, control structures, and statements. Within the host environment, JavaScript can be extended to include additional objects; for example, client-side JavaScript provides objects for controlling the browser and its Document Object Model (DOM).

This documentation provides a comprehensive overview of JavaScript fundamentals.

What you should already know

This guide assumes you have the following basic understanding:

  • A general understanding of the Internet and the World Wide Web (WWW).
  • Good working knowledge of HyperText Markup Language (HTML).
  • Some programming experience, if you want to understand the concepts better.

If you are new to programming, consider starting with simpler concepts before diving deep into JavaScript.

JavaScript and Java

JavaScript and Java are similar in some ways but fundamentally different in others. The JavaScript language resembles Java but does not have Java's static typing and strong type checking. JavaScript follows Java's expression syntax, naming conventions, and control-flow constructs, which was the reason why it was renamed from LiveScript to JavaScript.

In contrast to Java's compile-time system, JavaScript is a dynamic language that supports various programming styles, including object-oriented, imperative, and functional programming.

While they share "Java" in their names, they are distinct languages developed independently.

// Example of JavaScript code
let message = "Hello from JavaScript!";
console.log(message); // Example of Java code (for comparison)
public class MyClass {
  public static void main(String[] args) {
    System.out.println("Hello from Java!");
  }
}
Hello world

To get started with writing JavaScript, open the Scratchpad and write your first "Hello world" JavaScript code:

function greet() {
  console.log("Hello, world!");
}

Then, press Ctrl+R to see the output in the browser's console.

Variables

You use variables as symbolic names for values in your application. The names of variables, called identifiers, conform to certain rules.

A JavaScript identifier must start with a letter, underscore (_), or dollar sign ($). Subsequent characters can also be digits (0-9).

  • `var`: Declares a variable, optionally initializing it to a value.
  • `let`: Declares a block-scoped local variable, optionally initializing it to a value.
  • `const`: Declares a block-scoped, read-only named constant.
var x = 42;
let y = "Hello";
const PI = 3.14159;
Data types

The latest ECMAScript standard defines seven data types:

  • `Number`: e.g., `123`, `3.14`, `-5`.
  • `BigInt`: for arbitrary precision integers.
  • `String`: e.g., `"hello"`, `'world'`.
  • `Boolean`: `true` or `false`.
  • `null`: A special keyword denoting a null value.
  • `undefined`: A top-level property whose value is not defined.
  • `Symbol`: A data type whose instances are unique and immutable.

And `Object` for more complex structures.

let num = 10;
let str = "JavaScript";
let bool = true;
let obj = { key: "value" };
Functions

A function is a block of code designed to perform a particular task. A JavaScript function is executed when "something" invokes it.

Functions can be defined using `function` keyword, arrow functions, or as methods in objects.

// Function declaration
function add(a, b) {
  return a + b;
}

// Arrow function
const multiply = (a, b) => a * b;
Control flow

Control flow statements are used to control the order of execution of statements in a program. Common control flow statements include conditional statements and loops.

  • `if...else`: Executes different blocks of code based on a condition.
  • `switch`: Executes different blocks of code based on multiple possible values of an expression.
  • `for`: Loops through a block of code a number of times.
  • `while`: Loops through a block of code while a specified condition is true.
if (condition) {
  // code to execute if condition is true
} else {
  // code to execute if condition is false
}

for (let i = 0; i < 5; i++) {
  console.log(i);
}
